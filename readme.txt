ACF Address fields with GPS coordinates geocoding 

== Description ==

Adds ACF Address fields. The GPS coordinates are geocoded on save with OSM Nominatim.

= Compatibility =

This ACF field type is compatible with:
* ACF 5

== Installation ==

1. Copy the `acf-address-coordinates` folder into your `wp-content/plugins` folder
2. Activate the Address with Coordinates plugin via the plugins admin page
3. Create a new field via ACF and select the Address with Coordinates type
4. Read the description above for usage instructions
